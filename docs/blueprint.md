# **App Name**: SafeHive

## Core Features:

- Configurable Landing Page Display: Displays various sections including Navigation, Hero, Comparison, Technology, Deployment, and Footer content with their specified visual and textual elements.
- Real-time Safety Simulation Visualizer: Interactive visual canvas in the Hero section showing animated avatars (Humanoid Robot, Human Worker) with concentric safety rings, demonstrating proximity-based speed adjustments upon user interaction.
- Hive Concept Interactive Diagram: An interactive SVG/CSS graphic illustrating the 'Hive Concept', allowing users to toggle an 'Obstruction' to visualize dynamic data routing and Confidence Factor adjustments.
- Interactive System Configurator: A calculator widget with state management for inputting Workplace Area, Non-Native Machines, and Non-Native Humans. It dynamically calculates and displays a Bill of Materials (BOM) and total estimated hardware cost.
- AI-Powered Configuration Insights Tool: Analyzes the user's selected system configuration from the configurator and provides dynamic, tailored insights or recommendations for optimizing system efficiency and safety confidence factor based on the inputs.
- Quote Request System: A modal popup allowing users to export a detailed BOM and request a final quote by submitting their Name, Work Email, and Company Name.

## Style Guidelines:

- A dark scheme is chosen to convey a high-tech, clean, and professional industrial aesthetic. The primary color is a vibrant Safety Yellow (#F7C611), symbolizing caution and high visibility. The background color is a deep, warm slate-grey (#2C2A24), providing a strong contrast to the primary and safety accents. The accent color is a striking Orange-Red (#EF4E0A), reserved for critical alerts and calls to action.
- Headlines use 'Space Grotesk', a sans-serif with a techy and modern feel. Body text uses 'Inter', a clean, objective sans-serif ensuring high readability for technical content and data-rich tables.
- Utilize modern, clean, and geometric icons. Icons should visually represent industrial equipment, safety concepts, and connectivity. Custom avatars for 'Humanoid Robot' and 'Human Worker' will maintain a consistent high-tech brand image.
- A highly structured, grid-based layout emphasizing clear organization and readability. Key interactive sections, like the configurator and diagrams, will utilize a spacious, minimalist design. A two-column card layout will be used for comparisons, and timelines or horizontal grids for step-by-step processes, maintaining a professional and intuitive user experience.
- Subtle, functional animations will enhance interaction and user understanding. This includes smooth scrolling to internal anchors, fluid transitions for interactive elements like safety rings and robot movement, and dynamic visual feedback in the Hive Concept diagram. Animations will be purposeful, reflecting precision and control characteristic of industrial systems.
